# savecloyne.com

website for [savecloyne.com](http://savecloyne.com) built using [DocPad](http://docpad.org).

## how to

### install

```
git clone https://github.com/cloyne/savecloyne.com
npm install
```

### run

`npm start`

### develop

`npm run develop`

### deploy

`npm run deploy`
